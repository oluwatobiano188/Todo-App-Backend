# React ToDo App

A simple React ToDo app for learning DevOps workflows.

## Run locally
```bash
npm install
npm start
```

## Build for production
```bash
npm run build
```

## Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
